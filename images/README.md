# Site Images

Place your images here. Supported formats: `.jpg`, `.png`, `.webp`

## Folders

- `hero/` — Hero background image. Name it `hero-bg.jpg` (or .png/.webp). Recommended: 1920×1080px or wider.
- `languages/` — One image per language card. Name them: `runyankore.jpg`, `rukiga.jpg`, `runyoro.jpg`, `rutooro.jpg`. Recommended: 400×200px.
- `culture/` — Culture/about section image. Name it `culture.jpg`. Recommended: 800×500px.
- `team/` — Team member photos (for future use). Name them `member-name.jpg`.

All images are optional — the site falls back to gradients/placeholders if no image is found.
